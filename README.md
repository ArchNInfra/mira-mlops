# Rasalhague – MLOps Pipeline

Rasalhague implements a lightweight MLOps workflow built on GCP services,
providing a minimal setup for data ingestion, training, model registry,
and deployment automation.

## Scope
- Vertex AI Pipeline (training and evaluation)
- BigQuery-based data loading
- Model registry integration
- Optional deploy step targeting Cloud Run or an external inference service

## Stack
- Python (pipeline components)
- Google Cloud: Vertex AI, BigQuery, Artifact Registry
- Cloud Scheduler (optional)
